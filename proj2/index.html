Section II: Triangle Meshes and Half-Edge Data Structure

Part 3: Area-Weighted Vertex Normals

My implementation to get area-weighted vertex normals:

* A (unit) normal is a vector that is perpendicular to a face/vertex.

  1. Use the half-edge data structure to iterate through neighboring faces (triangles) of the given vertex, i.e., faces that have the given vertex as one of its vertices.
  2. For each triangle, get the area of that triangle by calculating the cross product of the two boundaries of that triangle.
  3. After looping through all neighboring triangles and getting their areas, you have the sum of the areas of all neighboring triangles. You can now normalize the sum to get an area-weighted normal.

Part 6: Loop Subdivision for Mesh Upsampling

My implementation for loop subdivision:
  1. Compute new positions for all the vertices in the input mesh, using the loop subdivision rule. (See the “Old vertex” graph above.)
    1. I implement a function, oldVertexNewPosition(old_vertex), and store the new position in old_vertex->newPosition. The updated position of an old vertex is: (1 - nu)*original_position + u*original_neighbor_position_sum
    2. Mark all vertices in the original mesh as old vertices by setting old_vertex->isNew = false.
  2. Compute the updated vertex positions associated with edges, using the loop subdivision rule. (See the “New vertex” graph above.)
    1. I implement a function, newVertexPosition(old_edge), and store the new position in old_edge->newPosition. The updated position of a new vertex splitting the shared edge (A, B) between a pair of triangle (A, C, B) and (A, B, D) is: 3/8 * (A + B) + 1/8 * (C + D)
    2. Mark all edges in the original mesh as old edges by setting old_edge->isNew = false.
  3. Split every edge in the mesh.
    1. Set the position of the new vertex properly.
    2. Set the isNew status properly.
      1. The new vertex: isNew=true.
      2. After splitting an edge, the edge will now be represented by 2 edges. These subdivided edges that come from splitting an edge in the original mesh should still be marked as old ones: isNew=false.
      3. After splitting an edge, 2 new edges will be created. These edges should be marked as new ones: isNew=true.
  4. Flip any new edge that connects an old and new vertex.
  5. The new vertex positions of old vertices were stored in old_vertex->newPosition. Now, update the into old_vertex->newPosition to final old_vertex->position.
